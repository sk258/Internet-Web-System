CSS Properties

  • Height :

    This gives you how much height you want of any block, image etc.

  • Width :

    This gives you how much width you want of any block, image etc.

  • Display :

    By using this property you can shows your content in block, inline, flex or combination of these like inline-block.

  • padding :

    for giving the space around any text, image, table or anything.

  • margin :

    for giving the space around any text, image, table or anything.

  • border :

    It provides border around anything.

  • background-color :

    Fill the background color.

  • background-image :

    sets the background image.

  • float :

    It aligns the elements.

  • color :

    Sets the color as you want.

  • color :

    Sets the color as you want.

  • font-family :

    Sets the the text in your style as you want.

  • text-align :

    It aligns the text in center, left or right.

  • font-size :

    sets the size of the text.
